# MoskvaVPN Landing Page

Simple static landing page for MoskvaVPN service.

## Deployment

This is a static HTML/CSS site. No build step is required.

- **Vercel**: Import project, select "Other" framework (or it should auto-detect static), override build command to `null` if needed.
- **Netlify**: Drag and drop the folder or connect repo. Build command is empty, publish directory is `/`.
